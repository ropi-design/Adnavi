# Gemini API 無料プランガイド

## ✅ 無料で使えます！

Gemini API は**完全無料**で利用でき、アカウント契約も不要です。

### 🎁 無料プランの範囲

**リクエスト制限**

-   1 分間: 60 リクエスト
-   1 日間: 1,500 リクエスト
-   1 ヶ月: 相当数のリクエスト（無料）

**トークン制限**

-   1 分間: 32,000 トークン

**初期クレジット**

-   $300 相当の無料クレジットが付与（約 12 ヶ月分）

### 🚀 無料で始める方法

#### ステップ 1: Google AI Studio にアクセス

https://makersuite.google.com/app/apikey

#### ステップ 2: 無料アカウントでログイン

-   Google アカウントでログイン（Gmail アカウントで OK）
-   **課金情報は不要**

#### ステップ 3: API キーを作成

1. 「Get API Key」または「Create API Key」をクリック
2. プロジェクトを選択（新規作成も可能・無料）
3. API キーが生成される

#### ステップ 4: API キーをコピー

生成されたキーを`.env`ファイルに貼り付け：

```env
GEMINI_API_KEY=あなたのAPIキーをここに貼り付け
GEMINI_MODEL=gemini-1.5-pro-latest
```

### 💰 料金について

#### 無料範囲内なら

-   \*\*完全無料
-   クレジットカード不要

#### 無料範囲を超えると

-   有料プランへの登録が求められます
-   $7/百万トークン程度から
-   Adnavi 程度の使用量なら問題ありません

### 📊 使用量の目安

**Adnavi での使用例**

-   1 レポート生成: 約 2,000-5,000 トークン
-   無料クレジット: $300 相当（約 60,000,000 トークン）
-   **約 12,000 回のレポート生成が可能**

### ⚠️ 注意事項

1. **API キーの管理**

    - 他人に共有しない
    - GitHub にコミットしない（.gitignore に含まれています）

2. **使用制限**

    - 大量リクエストを短時間で送らない
    - レート制限（1 分 60 リクエスト）に注意

3. **エンタープライズ利用**
    - 商用の本番環境では有料プラン検討
    - 無料プランは開発・個人利用向け
